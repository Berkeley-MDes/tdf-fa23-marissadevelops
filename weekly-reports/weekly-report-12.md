## Project 4 Proposal: Diagrammatic goal-setting AI for adult learners 

### Summary

I will be creating  a digital application of an analog goal setting activity I  built in a design-based research course; I would integrate my interview protocol into the zerowidth LLM, develop a simple interface to have students track their ideas, and then creating a visual output to represent the conversations they have with a guided AI.

### Research Plan

1. Train LLM on goal-setting activity curriculum and interview protocol using ZeroWidth
2. Utilize ZeroWidth API to output specific user data to a web application
3. Determine UX structure for interactive diagramming templates
Visualize select user data as a diagram using p5.js
